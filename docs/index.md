# 简介

* 进行自动的程序化股票交易
* 支持跟踪 `joinquant`, `ricequant` 的模拟交易
* 支持跟踪 雪球组合 调仓, 实盘雪球组合
* 支持通用的同花顺客户端模拟操作
* 支持命令行调用，方便其他语言适配
* 支持远程操作客户端
* 支持 Python3 , Linux / Win / Mac
